# study —— 我的学习笔记目录

这里是我跟随 `Quant-for-Beginners` 教程学习的个人笔记与调试代码。

## 目录结构

```
study/
├── README.md                 # 本文件
├── notes/                    # 每章一篇 Markdown 笔记（主战场）
│   ├── ch01_什么是量化金融.md
│   ├── ch02_你的第一个量化实验.md
│   ├── ch03_移动平均线策略.md
│   └── ch04_策略回测.md
└── scratch/                  # 调试用可运行 .py 脚本
```

## 用法

- **笔记**：在 `notes/chXX_*.md` 里记学习心得、公式推导、疑问、对原章节代码的逐段注释。
- **调试**：当光看笔记不够、需要真跑/打断点时，把代码拆到 `scratch/` 下的 `.py` 文件单步调试。
  - 调试脚本里引用数据用相对路径 `../data/...`（指向仓库根的 `data/`，不重复存数据）。
  - 运行：在仓库根目录 `python study/scratch/xxx.py`，或 `cd study/scratch && python xxx.py`。
