## 关键概念

### 1. OHLCV 五大字段

| 字段 | 含义 | 用途 |
|------|------|------|
| Open | 开盘价 | 当天第一笔成交价 |
| High | 最高价 | 当天成交的最高价 |
| Low | 最低价 | 当天成交的最低价 |
| **Close** | **收盘价** | **最常用，代表当天最终定价** |
| Volume | 成交量 | 当天成交的股数 |

### 2. 收益率（Return）

**直觉理解**：100元 → 110元，赚了10元，收益率 = 10/100 = 10%

**公式**：
```
r_t = (P_t - P_{t-1}) / P_{t-1}
```

**pandas 实现**：
```python
df['日收益率'] = df['Close'].pct_change()
```

### 3. 波动率（Volatility）

用**标准差**衡量价格波动剧烈程度：
```python
volatility = returns.std()  # 标准差越大 = 波动越大
```

## 疑问与思考

### Q1: 为什么用收盘价而不是开盘价？
收盘价代表一天最终定价，更能反映市场对该股票当日的整体评价。开盘价只反映第一瞬间。

### Q2: 标准差为什么能衡量波动？
标准差描述数据偏离平均值的程度。日收益率标准差大 → 每天涨跌幅度变化大 → 波动剧烈。

### Q3: 累计收益率的原理是什么？
每天按 `(1 + r)` 连乘：
- 第1天涨10%：100 × 1.1 = 110
- 第2天跌5%：110 × 0.95 = 104.5
- 累计收益率 = (104.5 - 100) / 100 = 4.5%

公式：`(1 + r1) × (1 + r2) × ... × (1 + rn) - 1`

### Q4: Histogram（直方图）是什么？
**定义**：按数据大小分成若干区间（bin），统计每个区间的数据数量，用柱子高度表示。

**在量化中的作用**：
```python
plt.hist(returns, bins=40)  # bins=40 表示分成40个区间
```

**能看出什么**：
1. **中间最高的柱子** = 最常见的涨跌幅
2. **两边的尾巴** = 极端涨跌的频率
3. **分布是否对称** = 涨跌是否均衡

### Q5: Histogram 尾巴很长说明什么？
**核心特征**：标准差大，波动剧烈 ✅

**具体含义**：
- 大部分时候涨跌幅在正常范围
- 但偶尔会出现**极端大涨或大跌**（黑天鹅事件）
- 例如：因突发新闻单日暴涨10%或暴跌15%

**投资含义**：
- **长尾巴（厚尾）**：科技股、成长股（特斯拉、英伟达）
  - 高风险高收益
  - 需要严格止损
- **短尾巴（正态分布）**：银行股、公用事业股（苹果相对稳定）
  - 波动平稳
  - 适合保守投资
