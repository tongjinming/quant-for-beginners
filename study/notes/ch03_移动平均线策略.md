## 关键概念

### 1. 趋势 vs 噪声

- **上涨趋势**：价格整体往上走，低点越来越高
- **下跌趋势**：价格整体往下走，高点越来越低
- **噪声**：短期上蹿下跳的"毛刺"，不代表大方向变化

量化策略的第一步：**把大方向（趋势）和每日乱动（噪声）分开**。

### 2. 移动平均线（Moving Average, MA）

**定义**：最近 n 天收盘价的算术平均

$$
MA_n(t) = \frac{1}{n} \sum_{i=0}^{n-1} P_{t-i}
$$

**特点**：
- n 越小 → 均线跟得越紧（更敏感，如 MA5）
- n 越大 → 均线越平滑（更迟钝但更稳，如 MA20）

**作用**：磨平噪声，看清趋势

### 3. 金叉 vs 死叉

| 名称 | 发生了什么 | 市场直觉 |
|------|------------|----------|
| **金叉** | 短期均线（MA5）从下往上穿过长期均线（MA20） | 短期变强，偏多/买入信号 |
| **死叉** | 短期均线（MA5）从上往下穿过长期均线（MA20） | 短期变弱，偏空/卖出信号 |

**注意**：金叉/死叉只是描述交叉现象，不等于"一定赚钱"——需要回测验证。

### 4. 双均线策略

**规则**：
```
如果 MA5 > MA20:
    持有 / 买入 (signal=1)
否则:
    空仓 / 卖出 (signal=0)
```

这是著名的 **Moving Average Crossover** 策略的最简版本。

### 2. 模拟三种市场状态

```python
np.random.seed(7)
n_up, n_down, n_noise = 60, 50, 70

ret_up = np.random.normal(0.004, 0.008, n_up)      # 上涨段
ret_down = np.random.normal(-0.005, 0.010, n_down) # 下跌段
ret_noise = np.random.normal(0.0, 0.015, n_noise)  # 横盘段

price = 100 * np.cumprod(1 + np.r_[ret_up, ret_down, ret_noise])
```

- `np.r_[]`：把三段收益率数组拼接成一条长数组
- `np.cumprod(1+ret)`：从起点 100 连乘，得到价格路径

### 3. 下载真实数据并计算均线

```python
raw = ak.stock_us_daily(symbol='AAPL', adjust='qfq')  # 前复权
raw['date'] = pd.to_datetime(raw['date'])
raw = raw.set_index('date').sort_index()
raw.columns = [c.capitalize() for c in raw.columns]  # 列名首字母大写

df = raw[['Close']].tail(500).copy()  # 取最近 500 个交易日

df['MA5'] = df['Close'].rolling(5).mean()
df['MA20'] = df['Close'].rolling(20).mean()
```

**注意**：
- akshare 返回的列名是小写，需要 `.capitalize()` 统一成 `Close`
- `rolling(N).mean()` 前 N-1 天是 NaN

### 4. 检测金叉/死叉

```python
df['spread'] = df['MA5'] - df['MA20']              # 短均线减长均线
df['cross'] = np.sign(df['spread']).diff()         # 符号变化

golden = df[df['cross'] > 0].dropna(subset=['MA5', 'MA20'])  # 金叉
death = df[df['cross'] < 0].dropna(subset=['MA5', 'MA20'])   # 死叉
```

**核心逻辑**：
- `spread = MA5 - MA20`：差值
- `np.sign(spread)`：取符号（正/负/零 → 1/-1/0）
- `.diff()`：差分，符号从负变正 → `cross > 0` → 金叉

### 5. 生成策略信号

```python
df['signal'] = (df['MA5'] > df['MA20']).astype(int)  # 1=持仓，0=空仓

df['trade'] = 0
df.loc[df['cross'] > 0, 'trade'] = 1    # 金叉日标记买入
df.loc[df['cross'] < 0, 'trade'] = -1   # 死叉日标记卖出
```

**关键**：`signal.shift(1)` 在回测时使用，避免偷看未来（本章先聚焦可视化）。

### 6. 可视化策略信号

```python
buys = df[df['trade'] == 1]
sells = df[df['trade'] == -1]

ax.scatter(buys.index, buys['Close'], marker='^', s=120, color='limegreen',
           edgecolors='darkgreen', linewidths=1, zorder=6, label='买入 ▲')
ax.scatter(sells.index, sells['Close'], marker='v', s=120, color='salmon',
           edgecolors='darkred', linewidths=1, zorder=6, label='卖出 ▼')
```

**专业量化图表要素**：
- 价格 + 双均线
- 绿色 ▲ = 买入点（金叉）
- 红色 ▼ = 卖出点（死叉）
- 下方色带 = 持仓状态（0/1）

### 任务 3：思考题

**Q1：金叉一定赚钱吗？**

❌ **不一定**。原因：
1. **假信号（震荡市）**：横盘时均线反复交叉，来回止损
2. **滞后性**：均线本质是"看后视镜"，真正趋势确立时可能已经涨了一大段
3. **没有考虑成本**：频繁交易的手续费和滑点会吞噬利润

**Q2：为什么第四章要做回测？**

因为：
1. **验证历史表现**：这套规则在过去 5 年真的赚钱吗？最大回撤多大？
2. **量化风险收益**：夏普比率、胜率、盈亏比等指标
3. **发现问题**：哪些时段策略失效？什么市况下表现差？
4. **避免过度自信**：看着图上几个金叉很完美，不代表整体有效

> **金叉只是信号，回测才是验证。策略没有经过历史检验，就是"看起来不错的想法"。**

## pandas API 踩坑

### `.last()` 方法已移除

**现象**：
```python
recent = df.last('6M')  # ❌ AttributeError: 'DataFrame' object has no attribute 'last'
```

**原因**：pandas 新版已移除 `.last()` 方法（用于选择时间序列最后一段）

**解决**：
```python
# 方案 1：直接用 tail()
recent = df.tail(120)  # 取最近 120 个交易日（约 6 个月）

# 方案 2：时间筛选（如果需要精确 6 个月）
recent = df[df.index >= df.index[-1] - pd.DateOffset(months=6)]
```

## 本章总结

本章学会了把"感觉均线要交叉了"变成 `if MA5 > MA20: signal=1` 的机器可执行规则，并用散点图标注买卖点——这就是量化策略的可视化起点。

**核心认知转变**：
1. **趋势 vs 噪声**：移动平均的本质是用"平滑"区分大方向和短期乱动
2. **规则化思维**：把主观判断变成可验证的代码逻辑
3. **信号 ≠ 盈利**：金叉死叉只是交叉现象，不是赚钱保证，必须回测验证

**技术要点**：
- `rolling(N).mean()`：pandas 核心函数，计算移动平均
- `np.sign().diff()`：检测符号变化，捕捉交叉点
- `scatter()` 标注买卖点：让策略图具备"专业量化报告"质感
