## 本章目标

- 理解"量化金融"到底在干什么
- 用一段模拟行情跑通：收益率 → 均线规则 → 回测对比 的完整链路
- 体会"规则策略 vs 随机买卖 vs 买入持有"的差别

## 关键概念

- **量化**：把投资决策写成明确的、可被计算机执行的规则，而不是凭感觉。
- **移动平均线（MA）**：N 日收盘价的算术平均，用来平滑短期波动、判断趋势方向。
- **净值曲线（NAV）**：把每日收益率 `(1+r)` 连乘，起点视作 1 块钱，看策略累计涨跌。

### 1. 模拟三段行情（涨→跌→恢复）

```python
n1, n2, n3 = 90, 40, 120
ret = np.r_[
    np.random.normal(0.0010, 0.010, n1),   # 阶段1：缓涨
    np.random.normal(-0.012, 0.015, n2),   # 阶段2：急跌
    np.random.normal(0.0012, 0.012, n3)    # 阶段3：恢复
]
price = 100 * np.cumprod(1 + ret)          # 起点100，(1+r) 连乘
```

- `np.random.normal(均值, 标准差, 个数)`：从正态分布采样日收益率。
- `np.cumprod(1+ret)`：把日收益率转成价格路径。等价于 `price[t] = price[t-1] * (1 + ret[t])`。
- 疑问：为什么用 `(1+ret)` 连乘而不是累加？→ 因为收益是"百分比复利"，必须连乘才符合真实价格演化。

### 2. 构造 DataFrame & 算日收益率

```python
df = pd.DataFrame({"close": price})
df["ret"] = df["close"].pct_change().fillna(0)
```

- `pct_change()`：`(今天 - 昨天) / 昨天`，即日收益率。第一天是 NaN，用 `fillna(0)` 补 0。

### 3. 量化规则：收盘价 > 20日均线 则持有

```python
df["ma20"] = df["close"].rolling(20).mean()
df["signal_quant"] = (df["close"] > df["ma20"]).astype(int)
```

- `rolling(20).mean()`：20 日移动窗口均值。前 19 天是 NaN。
- 信号：1 = 持有，0 = 空仓。

### 4. 对照组：随机买卖

```python
rng = np.random.default_rng(7)
df["signal_random"] = rng.integers(0, 2, size=len(df))
```

- 用独立的随机数生成器（种子也设 7），每天等概率 0/1。

### 5. 算各策略日收益 —— 关键：信号用昨天的

```python
df["ret_quant"]   = df["signal_quant"].shift(1).fillna(0)   * df["ret"]
df["ret_random"]  = df["signal_random"].shift(1).fillna(0)  * df["ret"]
df["ret_buyhold"] = df["ret"]
```

- **重点**：`signal.shift(1)` 把信号往后挪一天。因为"今天收盘看到信号，明天才能按今天收盘价买卖"——不能偷看未来。
- 这是本项目的统一约定（见 `CLAUDE.md`）。

### 6. 日收益 → 净值曲线

```python
for col in ["ret_quant", "ret_random", "ret_buyhold"]:
    df[f"nav_{col}"] = (1 + df[col]).cumprod()
```

- 同样是 `(1+r)` 连乘。起点为 1，最终值就是累计倍数。

### 7. 画图

```python
fig, axes = plt.subplots(2, 1, figsize=(11, 8), sharex=True)
# 上图：价格 + MA20
# 下图：三种净值曲线
plt.tight_layout()
plt.show()
```
